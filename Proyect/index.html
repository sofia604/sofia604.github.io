Image Segmentation

The camera captures de Image of the object in the workspace, transformed it to grayscale and then it's segmented and a mask is applied

Grouping

Using algorithms of unsupervised learning the image is grouped and sectioned. The number of sections will determine the average distance between grip points

Grappling Points

Using the information of each segment, the average of the pixels that make it up is taken to determine the representative point of the section, using the average of this, validating that said point is within the image.
